The explicit grid we've defined now covers 2 rows and all 3 columns. Items below the first two rows are laid out by the implicit grid. Note that items in the implicit grid are only as high as they need to be to fit their content.

All items in the row take the same size. So as the middle item grows in height to accommodate the extra content, the other grid items on the same row take the same height to preserve the grid structure.

This was hard to do in CSS before now.
